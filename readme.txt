Get & DisPlay Product Detail From Amazon
== Description ==
<p>TMD AFF, an amazing plugin for Amazon Affiliate Marketing, makes the process of getting products information from amazon.com much easier in one click by collecting necessary information using ASIN Code.</p>
<h3>Major features included in TMD AFF:</h3>
<ul>
<li>Getting Product Feature Image without any copyright conflict.</li>
<li>Affiliate Link with your business ID tags registered on Amazon</li>
<li>Updated Product Price with right time.</li>
<li>The most recent Number of customer reviews on Amazon</li>
<li>Accurate Product Rating by customers.</li>
</ul>
Plugin Home Page : http://tmduc.com/wordpress/tmd-aff-plugin-danh-cho-niche-aff-amazon.html
== Installation ==
<h4>From your WordPress dashboard</h4>
<ul>
1. Download TMD AFF from https://wordpress.org/plugins/tmd-aff/
2. Visit 'Plugins > Add New'
3. Select Upload Plugin/ Browse for TMD AFF/ Install
4. Activate TMD AFF from your Plugins page.
5. You will see TMD AFF appearing on the left side as other plugins.
</ul>
<h4>From WordPress.org</h4>
<ul>
<li>1. Download TMD AFF https://wordpress.org/plugins/tmd-aff/ </li>
<li>2. Upload the 'TMD AFF' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)</li>
<li>3. Activate TMD AFF from your Plugins page.</li>
<li>4. You will see TMD AFF appearing on the left side as other plugins.</li>
</ul>
<h4>User Guides:</h4>
<p>1. Simply click on TMD AFF after you active the plugin.</p>
<p>2. Fill out the require information: Amazon Access Key, Amazon Secret Access Key, Amazon Associate Tag and the Button Text (eg.: Check price on Amazon).</p>
Refer https://thirstyaffiliates.com/knowledgebase/retrieve-amazon-product-advertising-api-access-key-secret-key if you do not know how to get Amz Access Key and Amz Secret Access Key.</p>
<p>3. Click on Save Changes.</p>
<p>4. Go to your single product reviews, you will see the panel named TMD Azon. Click on “enable” and put the product ASIN Code and click on Preview.</p>
<p>5. You now have the accurate current information of product from Amazon.com</p>
